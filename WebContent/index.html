Instrucciones de uso:

Suposiciones sobre los datos sobre los que funciona la página: cada entidad (un panel solar) constituye un feature dentro de una feature collection en formato TopoJSON. Cada feature tiene las mismas propiedades: nombre de la compañí, identificador del panel, y 3 parímetros de medida.

Para VER PANELES:

  1. En la columna izquierda de la página, seleccionar un parámetro de medida y una o todas las compañías.
  2. Pinchar el botón VER PANELES
  3. Volver a pinchar cada vez que se hace una nueva selección.

Para BUSCAR.

Se busca de acuerdo con el nombre de la compañía y con el identificador numérico de cada panel.

  1. En la caja del buscador, introducir un caracteralfanumérico.
  2. NO está desarrollada la visualización. (Pero sí se recogen los parámetros de búsqueda de forma dinámica en los datos de la base de datos).
